# baekjoon_algorithm
백준 알고리즘 코테 연습

```python
import sys
inp = sys.stdin.readline

n=input()

m=sys.stdin.readline()

s=inp()

print("result1:", n)
print("result2:", m)
print("result3:", s)
```

3줄을 입력하고 실행하면 input()은 개행문자가 생략됨을 확인<br>
